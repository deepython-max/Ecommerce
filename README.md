# Django E-commerce Website

A fully functional eCommerce web application built using Django, allowing users to browse products, manage accounts, and simulate an online shopping experience.

---

## Features

* User Registration & Login 
* Product Listing 
* Add to Cart 
* Session Management
* Basic Order Flow

---

## Tech Stack

* Python
* Django
* SQLite
* HTML, CSS
